# 🏗️ BuildSafe – 

BuildSafe Mobile App:

Mobile Based Infrastructure Risk Assessor For Contractors Working Under Disaster Resilience Programs

BuildSafe is a mobile-first infrastructure assessment tool designed to improve disaster-resilient construction workflows. It enables contractors and field engineers to complete structural audits even under low-connectivity conditions and generate actionable, AI-powered risk insights in real time.

![BuildSafe App Poster](https://github.com/user-attachments/assets/3ed21019-9b73-46f0-836c-1e965929aff6)

BuildSafe successfully delivers four core components: 

1️⃣ Assessment Management with offline-capable digital forms and automatic GPS location capture.

2️⃣ Risk Scoring Engine generating contextualized 0-100 risk scores by combining structural compliance with live weather data.

3️⃣ Compliance Management allowing engineers to define standards in plain English.

4️⃣ AI-Based Recommendations providing actionable, cost-aware mitigation strategies. 
User testing validated that contractors could complete full assessments under less-than-ideal network conditions, with all data reliably saved and synchronized upon reconnection.

⚙️ Step-by-Step Setup Guide
🪟 For Windows:

Download Ollama

Go to 👉 https://ollama.com/download

Download the Windows installer (.exe file).

Install

Double-click the installer and follow the prompts.

It installs Ollama as a background service.

Open Terminal (CMD / PowerShell)

Run this to check if Ollama works:

ollama --version

<br/>
Run your first model

Example: run Llama 3 (by Meta)

ollama run llama3

<br/>
It’ll download the model (~4–8 GB) and open a chat prompt.

Stop Ollama

Just close the terminal, or type exit.
